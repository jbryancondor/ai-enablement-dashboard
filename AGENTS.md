# AI Enablement Dashboard

## Tech Stack
This project is built using the following technologies:

* **React**: Core UI library.
* **Vite**: Build tool and development server.
* **TypeScript**: Primary programming language for type safety.
* **Recharts**: Data visualization library.
* **Lucide React**: Iconography.
* **Standard CSS**: Used for custom layouts.

## Visualization Options (Under Evaluation)
The following graph types are being evaluated to represent squad metrics. The final metrics and their visual representations will be determined from scratch.

### 1. Categorical Comparison
Graphs for comparing quantities across different categories (especially useful for long labels):
* Horizontal Stacked Bar Chart
* Grouped Bar Chart
* Heatmap Matrix

### 2. Goal and Progress Tracking
Graphs for displaying a current metric against a target goal:
* Target Progress Bar (Bullet Graph)
* Radial Gauge / Speedometer
* Standard Progress Bar

### 3. Workflow and Qualitative Data
Layouts for tracking tasks, status, and grouping by theme:
* Swimlane Matrix (Grouped Kanban)
* Flat Data Table
* Sankey Diagram / Network Node Graph
